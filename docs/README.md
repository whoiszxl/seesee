### 项目介绍
seesee，看看视频。基于DDD领域驱动设计的SpringCloud Alibaba短视频项目，移动端使用Flutter2.x构建，管理后台使用Vue 3.0 + Element Plus 进行构建，后端采用Hadoop与Flink等大数据框架构建实时计算与离线计算体系。
